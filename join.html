关于我们

Red Box 是一个致力于解决 月经贫困（Period Poverty） 的公益组织。随着组织不断发展，我们已经逐渐成长为一个覆盖全球的社区。我们相信，每一个人都能为推动经期平等贡献力量。

为什么加入 Red Box？

在 Red Box，你不仅能为公益事业贡献力量，还能结识一群志同道合的伙伴，把爱与能量传递到世界的每一个角落。我们期待更多小伙伴的加入，一起汇聚力量，推动改变！

同时，你也将在活动中不断成长：

  • 提升 领导力、组织力与沟通协调能力
  • 通过亲身参与 策划、执行与宣传 等环节，全面锻炼与提升自我
  • 在实践中积累宝贵经验，增强综合竞争力

招募岗位

本次开放以下四类岗位申请：

  • Red Box 组织成员 —— 一起参与公益项目，共同成长（若本校尚无校园代表，请申请校园代表）。
  • Red Box 地区代表（以省/州为单位） —— 代表所在地区，推动项目在本地的落地与执行；统筹和协调地区整体事务安排；扩大 Red Box 在本地区的影响力和社会声量。
  • Red Box 校园代表（小学/初中/高中/大学） —— 在校园内传播 Red Box 的公益理念，提升同学们的认知与关注；策划并组织校园相关活动；推动校园内的志愿者招募和团队建设。
  • Red Box 主席团成员（仅限现有成员） —— 负责统筹管理和战略规划。

申请流程

  • 所有岗位需填写 申请表
  • 组织成员：通过申请表审核后，结果将以邮件通知
  • 地区代表，校园代表，主席团成员：通过申请表审核后，结果将以邮件通知，并设有二轮面试环节

我们相信

每一位申请者都是独一无二的存在。无论你来自哪里，我们都期待你的加入。让我们在 Red Box 的大家庭里携手，为解决月经贫困贡献自己的力量，做一些真正有意义的事情。Breaking the silence, ending the poverty! 让我们不再保持沉默！
